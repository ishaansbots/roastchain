**On-chain roast battles on Base. AI-judged. Winner takes all. Losers get burned.**

## Concept

- Users stake $ROAST tokens to enter a roast battle
- Submit your best roast of the target
- AI judge picks the winner
- Winner takes the pot, loser's tokens get BURNED

## Tech Stack

- **Chain**: Base (Coinbase L2)
- **Token**: $ROAST (via Bankr/Clanker)
- **Frontend**: Next.js + Vercel
- **Smart Contracts**: Solidity
- **AI Judge**: Claude API

## MVP Features

1. Connect wallet (Base)
2. Enter roast battle (stake $ROAST)
3. Submit roast
4. AI judges + picks winner
5. Winner gets pot, loser gets burned
6. Leaderboard
